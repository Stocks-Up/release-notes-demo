# Notes de version 0.3.0

### Gérez vos notifications en toute liberté

Cette version vous offre un espace dédié pour définir précisément comment et quand vous souhaitez être contacté par Kulu — par canal, par catégorie, en un seul endroit.

**Nouveautés**

🔔 **Préférences de notifications**
• Définissez vos préférences individuellement pour les notifications par e-mail, Slack et dans l'application
• Contrôlez les alertes par catégorie : mises à jour de versions, résumé hebdomadaire et alertes de sécurité
• Vos paramètres de notification par défaut sont désormais inclus dans votre profil utilisateur

---

**Corrections de bogues et améliorations**
• Les paramètres de notification sont désormais appliqués de manière cohérente sur tous les canaux
