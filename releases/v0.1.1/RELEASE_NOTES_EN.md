# Release Notes 0.1.1

### A new way to explore African markets

Kulu now includes a comprehensive reference table of all 54 African countries, making it easier to browse key data at a glance.

**What's New**

🌐 **African Countries Reference**
• Interactive table with all 54 countries, capitals, and population data
• Filter by region — North, West, East, Central, and Southern Africa
• Search by country or capital name instantly
• Sort by any column for quick comparisons
