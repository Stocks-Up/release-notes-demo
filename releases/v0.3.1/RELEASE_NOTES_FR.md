# Notes de version 0.3.1

### Connexion Gmail simplifiée

Connecter votre compte Gmail devient plus simple. Cette mise à jour améliore le processus de configuration des identifiants pour que vous puissiez démarrer sans incertitude.

**Nouveautés**

🔐 **Intégration Gmail**
• Indications plus claires lors de la configuration de vos identifiants Gmail
• Messages d'erreur plus précis pour résoudre rapidement les problèmes de connexion
• La validation de la configuration confirme désormais que vos identifiants fonctionnent avant de les enregistrer

---

**Corrections de bogues et améliorations**
• Correction d'un problème où les erreurs d'identifiants Gmail n'étaient pas clairement signalées lors de la configuration
